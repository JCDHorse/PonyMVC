# Homemade MVC framework in PHP
Simple implementation of the Model View Controller (MVC) architectural pattern in PHP. This framework provides a basic structure for building web applications by separating the application logic, user interface, and data management.

## Features
- Simple routing mechanism
- Basic controller and view handling
- Easy to extend and customize
- Lightweight and minimalistic
- Docker support for easy deployment

## Starting
### Requirements
- Docker
- A computer
- Some basic knowledge of PHP and web development

### Launching the project with Docker
```bash
docker-compose up --build -d
```
